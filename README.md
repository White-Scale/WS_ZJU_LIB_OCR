##中文 
###背景
起因是室友提出的图书馆验证码构造简单（确实如此）  
本想合作一起完成，最后还是各做各的； 
###简介
使用jQuery对网页中按钮进行追踪；  
使用cavnas画板实现服务器中动态变化的图片的本地保存以及二值化；  
以数组的形式进行数据处理分析；  
识别思路将图片分块，将每块数字区域分成小块统计各小块的像素数，  
得到一组特征值数组，并与给出的各数字特征值进行比对完成识别；  
分割算法为直接分割，需要优化；
